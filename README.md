## 📋 How to use the Kanban board

OpenMac uses a Kanban flow with cards and columns. A simple setup may include:

- Backlog
- To Do
- Doing
- Review
- Done

To move work through the board:
1. Create a task card
2. Add a short title
3. Write the task goal
4. Place the card in Backlog or To Do
5. Move it to Doing when work starts
6. Send it to Review when it is ready
7. Mark it Done when finished

You can also:
- add notes to cards
- set labels for task type
- split large tasks into smaller steps
- assign one task to one person or one agent
- keep related tasks in the same project
